HTML & Web Principles Course

HTML is the foundation upon which web pages and emails are constructed and is the best place to start if you’re diving into web design or web development.

Even if you plan to use a web authoring tools that generates the HTML code automatically, having a solid understanding of HTML and some CSS makes learning those systems much easier and ensures that you can work with the code directly to fix any problems that may arise.

To view the full course outline, please visit HTML & CSS Courses at The Web School.
