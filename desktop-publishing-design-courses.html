Desktop Publishing Design Courses

Not sure what each Adobe application can do?

InDesign

Create everything from a flyer to a book or annual report. Adobe sInDesign allows you to easily layout a page with powerful image and text formatting controls. Use Master Pages, Text and Object Styles to automate and save time.
See full InDesign course outline and video.

Photoshop

Is an image too dark or need cropping? Need to remove a background from a photo or retouch images to remove blemishes? If you need to alter a photo in any way or create a composition or design, Adobe Photoshop is what you'll need.
See full Photoshop course outline.

Illustrator

Create logos, diagrams, maps and infographics with ease. By natively creating vector graphics, content created with Adobe Illustrator can generally be scaled to any size from a business card through to a billboard.
See full Illustrator courses and video.
