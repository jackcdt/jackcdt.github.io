WordPress Advanced Course

This course primarily involves teaching students how to create their own theme in WordPress and expands upon the skills taught in WordPress Essentials. Topics that are discussed in conjunction with themes include custom functions, code snippets, adding and customising plugins and search engine optimisation (SEO) considerations.
