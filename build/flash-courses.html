Adobe Authorised Certificate Courses

Adobe Flash can be used to create web banners, interactive projects, elearning material and entire websites. If you need more than basic interactivity, we also offer Flash ActionScript training and courses. City Desktop has one of the very few Adobe Certified Flash instructors Australia wide. While we recommend using the latest version - Flash CC, we also have previous versions available.

To produce animation & interactivity that works on all devices, including iPads & iPhones, see Adobe's new Edge Animate.
