Dreamweaver & CSS Courses

Dreamweaver is the industry standard web authoring package. From laying out pages to uploading and maintaining sites, Dreamweaver offers intuitive tools that make it easy to get up to speed with web design. While Dreamweaver uses a WYSIWYG interface to create your website, it is essential to have a solid understanding of HTML to grasp the many topics covered in the 2 day Dreamweaver course. Attend courses for Dreamweaver CC, CS6 or CS5.
