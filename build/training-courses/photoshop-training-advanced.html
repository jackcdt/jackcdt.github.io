Special Effects

  • Emphasis is on understanding the process and how to use a combination of functions to generate the effect. The following is a brief description of the types of effects that will be covered:
  • Creating editable type that appears collated through the use of Clipping Masks
  • Fade two or more images together smoothly, and combine colour and b/w into the same image using Layer Masks
  • Explore the use of filters and using other images in a Layer Mask
  • Combining vector shapes with raster images to create interesting covers and designs
  • Embossing editable text out of a textured scan, or carving a logo into a scan using Layer Styles. Then automating this process

Functions used

  • The emphasis will be on exploring various approaches to creating special effects and many of the special features of the program will be used. Such as the use of Layers, Layer Masks, Layer Styles, Adjustment Layers, Vector Shapes, Blending Modes and Clipping Masks. Creating Animated Gifs. Creating, manipulating, saving, applying and feathering selections. Using Styles, History and keyboard shortcuts to simplify processes and applying several key filters – Gaussian Blur, Wave, Ripple, etc. There is an emphasis on using shortcuts and automation within Adobe Photoshop and setting the right Application preferences.
