InDesign Long Documents Course

Is your goal to publish long documents such as books or annual reports with ease? This course focuses on essential techniques that will help you set up your long documents properly from the start and which will also help with those inevitable changes in the most painless way possible.

Achieving a successful and easy long document workflow in Adobe InDesign is all about planning and execution. Long documents are the most complex things you can build with InDesign. This course shows you how to get excellent results quickly and boost your efficiency through showing you how to build long documents the right way from the ground up, using InDesign’s tools and automation features.

Our instructors are Adobe Certified, industry experienced and passionate about training! We recommend using the latest version - InDesign CC (Creative Cloud), but we also have CS6 and CS5 options. Not sure which version? See our article on Adobe software versions.

At City Desktop Training – an Adobe Authorised Training Centre – upon course completion participants receive a Certificate endorsed by Adobe.

Based On Styles

  • Basing Styles on a “parent” style is a powerful cascading technique that will help create styles quickly as well as change styles quickly and easily. Use one style as the basis for new variations. InDesign’s Based On feature works with ANY styleable element including Paragraph, Character, Object, Table and Cell.

GREP Styles

  • GREP styles enable you to apply dynamic pattern-based character styling to your text. Create GREP styles as part of a paragraph style. InDesign searches for that pattern and applies the character style whenever it finds a match.

Nested Styles

  • Nested styles allow you to apply dynamic character styling to a continuous range of text. Using Nested styles allows you to tell InDesign which character styles to apply and which characters mark the end of a range. Repetitious application of styling based on patterns can be achieved easily by utilising InDesign’s Nested Styles to repeat any sequence of styles.

Text Variables

  • Text variables are clones of text or data taken from elsewhere in InDesign. They can be used to automatically change according to the context in which they appear.

Cross-References

  • InDesign's cross-referencing allows you to create dynamic links between text anchors or paragraphs of styled text. Automatically generate text elements and elements that can be easily updated with the Hyperlinks panel. Learn to control the structure and formatting of elements by defining cross-reference formats.

Table of Contents

  • Table of Contents is one of InDesign’s powerful long document features. Master InDesign’s TOC panel and get it working for you.

Indexing

  • Learn how to use InDesign’s extensive Index features. Practical and powerful the Index panel built in to InDesign will assist and speed you through the index creation process.

The Book Panel

  • Manage long document split over multiple InDesign files. Organise, synchronise and output these files all as one using InDesign's Book panel.
