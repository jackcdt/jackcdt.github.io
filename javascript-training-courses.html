JavaScript & jQuery Training

JavaScript is essentially the scripting language of the web and is used to make websites more interactive beyond the standard static pages designed in HTML. Some of the examples of JavaScript's functionality include the following: Manipulating Images, Using Cookies, Drop Down Navigation Systems, Form Validation and Processing, Calculations, Changing the Content and Styling of Web Pages and Games.

jQuery is a cross-browser JavaScript library designed to simplify the client-side scripting of HTML with JavaScript and is an excellent follow-on from the 2 day introduction to JavaScript course.
